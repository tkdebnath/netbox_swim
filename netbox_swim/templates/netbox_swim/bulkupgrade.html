Execution Routing

The system will process each device and execute them in parallel if they possess:

  • An IP Address for SSH connections
  • A structurally bound Hardware Group
  • A bound Workflow Template tied to that Group
  • A valid Target Golden Image
Maintenance Window Scheduling

Use the "Schedule For" field to defer all upgrade jobs to a future maintenance window.

  • Blank = Jobs execute immediately upon submission
  • Future datetime = Jobs are created with Scheduled status and executed automatically at the specified time via the RQ scheduler

Scheduled jobs can be cancelled by deleting the UpgradeJob record before the scheduled time. The datetime is interpreted in the server's timezone.
